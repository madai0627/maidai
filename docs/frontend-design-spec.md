## 1. 设计系统概述 (Design System Overview)

### 1.1 设计原则

1. **用户至上** - 每个设计决策都服务于用户需求
2. **简洁清晰** - 减少认知负担，信息层次分明
3. **一致性** - 统一的视觉语言和交互模式
4. **可访问性** - 符合 WCAG 2.1 AA 标准
5. **响应式** - 适配各种设备和屏幕尺寸

### 1.2 设计目标

- 提升用户对系统功能的认知完整度
- 减少操作步骤，提高使用效率
- 增强系统的专业感和一致性
- 确保良好的可访问性和响应式体验

## 3. 字体系统 (Typography)

### 3.1 字体族

- **中文字体**：`-apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif`
- **英文字体**：`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif`
- **等宽字体**：`"SF Mono", Monaco, "Cascadia Code", "Roboto Mono", Consolas, "Courier New", monospace`

### 3.2 字体大小

| 级别 | 大小 | 行高 | 字重 | 用途 |
|------|------|------|------|------|
| H1 | 28px | 40px | 600 | 页面主标题 |
| H2 | 24px | 34px | 600 | 区块标题 |
| H3 | 20px | 28px | 500 | 卡片标题 |
| H4 | 18px | 26px | 500 | 小标题 |
| Body Large | 16px | 24px | 400 | 正文（大） |
| Body | 14px | 22px | 400 | 正文（常规） |
| Body Small | 12px | 18px | 400 | 辅助文字 |
| Caption | 12px | 16px | 400 | 说明文字 |

### 3.3 字体使用规范

- **标题层级**：最多使用 3 级标题（H1-H3）
- **最小字号**：移动端 ≥ 14px，桌面端 ≥ 12px
- **行高**：正文行高 ≥ 1.5，标题行高 ≥ 1.2

### 4.1 基础单位

使用 **4px** 作为基础间距单位，所有间距为 4 的倍数。

### 4.2 间距量表

| 名称 | 大小 | 用途 |
|------|------|------|
| XS | 4px | 紧密元素间距 |
| SM | 8px | 相关元素间距 |
| MD | 16px | 常规元素间距 |
| LG | 24px | 区块间距 |
| XL | 32px | 大区块间距 |
| XXL | 48px | 页面区块间距 |

### 4.3 间距使用规范

- **卡片内边距**：16px（MD）
- **卡片间距**：20px
- **区块间距**：24px（LG）
- **页面边距**：20px（移动端），24px（桌面端）

### 5.2 阴影使用规范

- **卡片默认**：Level 1
- **卡片 hover**：Level 2
- **下拉菜单**：Level 2
- **模态框**：Level 3

## 7. 组件规范 (Component Specifications)

### 7.1 按钮 (Button)

#### 尺寸

| 尺寸 | 高度 | 内边距 | 字体大小 |
|------|------|--------|---------|
| Large | 40px | 12px 24px | 16px |
| Medium | 32px | 10px 20px | 14px |
| Small | 24px | 8px 16px | 12px |

#### 类型

- **Primary**：主色背景，白色文字
- **Default**：白色背景，主色边框
- **Success**：成功色背景，白色文字
- **Warning**：警告色背景，白色文字
- **Danger**：危险色背景，白色文字
- **Text**：无背景，主色文字

#### 状态

- **Normal**：默认状态
- **Hover**：背景色加深 10%，阴影 Level 1
- **Active**：背景色加深 20%
- **Disabled**：透明度 50%，禁用交互

### 7.2 卡片 (Card)

- **边框**：无（使用阴影代替）

## 9. 交互规范 (Interaction Specifications)

### 9.1 动画时长

| 类型 | 时长 | 缓动函数 | 用途 |
|------|------|---------|------|
| 快速 | 150ms | `ease-out` | 按钮点击、hover |
| 常规 | 300ms | `ease-in-out` | 页面切换、展开 |
| 慢速 | 500ms | `ease-in-out` | 复杂动画、过渡 |

### 9.2 页面切换动画

- **类型**：淡入淡出（Fade）
- **时长**：200ms
- **缓动**：`ease-in-out`

```css
.fade-enter-active,
.fade-leave-active {
  transition: opacity 0.2s ease-in-out;
}

.fade-enter-from,
.fade-leave-to {
  opacity: 0;
}
```

### 9.3 微交互

#### 按钮点击

- **效果**：轻微缩放 `scale(0.98)`
- **时长**：100ms

#### 卡片 Hover

- **效果**：阴影加深 + 轻微上移 `translateY(-2px)`
- **时长**：200ms

#### 数据更新

- **效果**：数字滚动动画
- **时长**：500ms

## 11. 可访问性规范 (Accessibility)

### 11.1 键盘导航

- **Tab 顺序**：逻辑顺序，从左到右、从上到下
- **焦点可见**：焦点状态明显（2px 主色边框）
- **快捷键**：
  - `Esc`：关闭模态框、下拉菜单
  - `Enter`：激活按钮、提交表单
  - `方向键`：导航列表、下拉菜单

### 11.2 屏幕阅读器

- **语义化 HTML**：使用正确的 HTML 标签
- **ARIA 标签**：为复杂组件添加 ARIA 属性
- **替代文本**：所有图片提供 alt 文本
- **表单标签**：所有输入框关联 label

### 11.3 颜色对比度

- **正文文字**：对比度 ≥ 4.5:1
- **大号文字**（18px+）：对比度 ≥ 3:1
- **UI 组件**：对比度 ≥ 3:1

### 11.4 触摸目标

- **最小尺寸**：44x44px（移动端）
- **间距**：触摸目标之间 ≥ 8px

## 12. 加载状态 (Loading States)

### 12.1 骨架屏 (Skeleton)

用于数据加载时的占位：

```scss
.skeleton {
  background: linear-gradient(
    90deg,
    #f0f0f0 25%,
    #e0e0e0 50%,
    #f0f0f0 75%
  );
  background-size: 200% 100%;
  animation: loading 1.5s ease-in-out infinite;
}

@keyframes loading {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}
```

### 12.2 加载指示器

- **全屏加载**：居中显示加载动画
- **局部加载**：在对应区域显示加载状态
- **按钮加载**：按钮内显示 loading 图标

## 13. 错误状态 (Error States)

### 13.1 错误提示

- **位置**：表单字段下方或页面顶部
- **样式**：红色文字 + 错误图标
- **内容**：清晰的错误说明 + 解决建议

### 13.2 空状态 (Empty States)

- **图标**：友好的空状态图标
- **文案**：说明为什么是空的 + 引导操作
- **操作**：提供主要操作按钮
